## Installation

### Relit
- Fork this [Repl](https://replit.com/@Jnx03/AqrMusicBox?v=1)
- Click on run button
- Done

### Xampp
- Download this [code](https://github.com/JNDEV03/AquariusMusic/archive/refs/heads/main.zip)
- put the file in htdocs
- click on start on apeche
- Done
